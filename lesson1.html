The Basic of the Web and HTML

1.万维网

The basic of the web，我们日常生活中遇到的网页都是由下面四部分组成的。

  • the web
  • html
  • urls
  • web application

world wide web(万维网)的主要文件类型是HTML(HyperText Markup language,超文本标记语言)。我们可以在网络上找到各种各样的文件，例如视频，图片，音乐等等。

万维网的主要组成部分如下图所示

￼
  • A:客户端，在电脑上运行用于显示web的文件的程序，例如浏览器
  • B:互联网，世界上最大的计算机网络
  • C:Http协议，web的主要协议
  • D:服务器，用于储存Web的组成文件的计算机

2.HTML基础

HTML标签简介

HTML由以下几部分组成

  • 文本内容
  • 标记语言
  • 另外的一些文档饮用，例如图像视频等等
  • 指向其他页面的链接html标签类似的结构都是由开始标签，结束标签以及标签中间的内容组成，一组标签以及其中的内容可以成为一个元素

HTML的常见标签

  • 粗体标签，<b>粗体</b>，
  • 斜体标签，<em>斜体</em>表示强调
  • a标签，<a href="www.reddit.com"> go to reddit</a>,点击 a标签中间的内容将会跳转到href属性中所写的网站
  • img标签，<img src="" alt="text">,img标签没有结束标签，被称为void tag，img标签中出现了HTML属性src和alt，src中存放该图片的路径，如图片没有正常加载则显示alt标签中的文字
  • br标签，<br>添加在文本后面可以换行
  • p标签，<p>段落</p>表示段落
  • span标签，<span></span>内联元素
  • div标签，<div></div>块状元素

可以去W3C查看更多的标签。

内联元素和块状元素

内联元素：Inline Elements，和相邻的内联元素会在同一行，而br标签就是结束这一行，例如b，em，img

块元素：Block Elements，总是独占一行，后面的元素需要另一行，有一个看不见的box，有自己的宽度和高度，例如p标签

HTML文档结构

￼

首行代码是创建的每个网页中都应该包含的内容。<!DOCTYPE html>定义了文档类型，这样，浏览器便可以正确显示网页。

文档从 <html> 标签开始。所有在网页中呈现的内容都要放在这个标签内。

<head> 标签定义了网页的元数据。它包含网页标题（就是出现在标签页顶端或浏览器窗口顶端的标题）。请注意，<head> 标签之所以被缩进到了新的层级是因为 <html> 标签是打开的，还没有关闭（新内容开始了，但还没有结束）。同样，<title> 标签被缩进到再下一个层级（因为 <head> 标签是打开的，还没有关闭）。

如果标签中的内容较短，我们通常的做法是，<title> 标签在同一行打开和关闭。另一方面， 标签在另一行关闭，结束标签与其开始标签保持同样的缩进层级。

标签的内容，代码结构中有没有让你感到很奇怪的地方呢？请注意，在 标签内，你有网页的主体内容。你可以看到在 <body> 内部缩进的 <h> 标签，以及与 <h> 标签相同缩进级别的 <p> 标签。这是一个重要的区别。标题和段落都是主体内容（ <body> 标签内部）的子元素。
