Creating a Structured Document With HTML

1.html,css,javascript在构建网页中的作用

  • html控制网页的整体结构
  • css控制网页的样式（外观）
  • javascript控制网站的功能

2.HTML-CSS-DOM

  • HTML - 超文本链接标示语言 – 创建网页时所用的标准标示语言。
  • CSS - 层叠样式表 – 用于描述用标示语言编写的一份文档的外观与格式的一种样式表语言。
  • DOM - 文档对象模型 – 一种跨平台且与语言无关的约定，用于在 HTML （及其它标示语言）中代表对象和与对象交互。每个文档的节点都是以树状结构组织的，称为 DOM 树。关于树状结构可以参考wiki中的相关文档。

3.无处不在的方框

要从方框的角度来看一个网页，将要制作的的网页页面分割成框(Boxify)，由大框逐步分割成小框。用div来定义方框，然后添加占位符，用class属性标记，以便于稍后应用，类似于在盒子上贴标签。
