Adding CSS Style to HTML Structure

CSS 是用于控制 HTML 元素样式的代码，全称为Cascading Style Sheets(层叠样式表)，在规则层级中逐层深入，知道你找到某个元素的具体的规则，然后对那个元素采用这个规则。

1.CSS样式

基本格式如下，选择器的名称，选择器分为元素选择器，类选择器和id选择器，id是唯一的。

          
          .选择器 {
            attribute: value
            }

每个浏览器都会有默认样式表来确定如何显示HTML元素。

  • WebKit (Chrome 及 Safari)
  • Firefox
  • Internet Explorer

因为这些规则在各个浏览器各不相同，所以提升不同浏览器之间的样式一致性需要很多工作。一种流行的解决方案是采用被称为 CSS 重置的方式，例如 normalize.css

2.CSS的优先级别

我们可以在不同地方来定义CSS样式，优先级别从上往下，越下面优先级越高。并且CSS的样式不仅仅只会作用于直接匹配的元素，也适用于被匹配元素的子元素。

  • 浏览器的默认样式（不同浏览器样式稍有不同）
  • 单独文件中的样式表（这是你将在大多数时候使用的）
  • HTML 中的样式表（在小型实战项目中可以这么做，但是不理想）
  • 一个元素中的内联样式表（虽然也可以做，但应该避免）

3.不同方式来定义CSS

方法 1：在 HTML 的 中编写 CSS

此方法适合小项目（例如在 scratchpad.io 中运行的小项目）。为此：

在 HTML 文档的顶部添加 <head> 元素。
在 <head> 元素内，添加 <style> 元素。
在 <style> 元素内写 CSS，然后将网页剩余部分的 HTML 放在下面。
例如：

          
        <head>
          <style>
            div {
              background-color : red;
            }
          </style>
        </head>
        <body>
          <div>
            This will have a red background.
          </div>
        </body>

方法 2：将 HTML 链接到单独的 CSS 文件

这样就再需要一个步骤，但是如果要完成的是大型项目，代码就更有组织性。为此：

将所有的结构性 HTML 写到一个文件里（称其为 main.html）。
将所有的 CSS 代码放到单独的文件里（称其为 main.css）。
向 HTML 的顶部添加 <head> 元素。
在 <head> 元素里添加 <link> 标记。因为 <link> 是“void tag”，所以你不需要添加结束 </link>。
向 <link> 添加以下属性：

rel="stylesheet"
        href="main.css"

例如，你可能会有一个如下所示的 main.html 文件：

          
        <head>
           <link rel="stylesheet" href="main.css">
        </head>
        <body>
          <div>
            This will have a red background.
          </div>
        </body>

在同一文件夹内，可以有一个如下所示的
main.css 文件：

          
    div {
          background-color : red;
        }

方法 3：用 HTML 内嵌地编写样式

这通常是一个糟糕的方法，因为这样会导致大量重复代码。但是你可能会见到使用这一方法的代码，所以有必要熟悉这一方法.
对于此方法，你需要修改每个 HTML 元素的样式属性。例如：

          <body>
          <div style="background-color: red; color: white">
            This div will have a red background and white text.
          </div>
          <div style="background-color: red; color: white">
            So will this one.
          </div>
          <div style="background-color: red; color: white">
            Now, what if I change my mind?
          </div>
          <div style="background-color: red; color: white">
            I'd rather have a black background...
          </div>
          <div style="background-color: red; color: white">
            Never do this!
          </div>
        </body>

4.CSS盒模型（BOX-MODEL）

html中的元素都是方框，由以下四个主要属性：

  • margin 外边距
  • border 边框
  • padding 内边距
  • content 内容
￼

由于每个方框都有太多的属性，所以有时候很难设置正确的方框尺寸，有两种技巧可以解决尺寸问题。

  • 根据百分比而不是像素来设置尺寸
  • 将每个元素的box-sizing属性设置为border-box，这样做会让元素的尺寸=内容+内边距+边框

5.元素的位置

  • Div 是块元素（对内联元素相对应)，所以默认地占满整个页面宽度。
  • 向相应的 CSS 添加 display: flex; 规则将覆盖此行为，使得 div 相互并排显示。
